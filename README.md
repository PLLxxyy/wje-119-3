# 马拉松赛事报名平台

全栈马拉松赛事报名平台，支持赛事浏览、在线报名、成绩上传和管理后台。

## 功能简介

- **赛事浏览**：首页展示即将开赛的赛事列表，每张卡片显示赛事名称、城市、日期、项目类型（全马/半马/亲子跑），带报名截止倒计时
- **赛事详情**：查看路线描述、起终点、关门时间、报名费和参赛物资说明
- **在线报名**：选择项目（全马/半马/亲子跑），填写个人信息和紧急联系人提交报名，支持模拟付款
- **个人中心**：查看已报名赛事、参赛号码，赛事结束后上传完赛证书和成绩
- **管理后台**：发布/编辑赛事、查看报名人数和选手数据统计、导出选手名单和物资发放记录（CSV）

## 技术栈

- 前端：Vite + React 18 + TypeScript + React Router v6
- 后端：Express + TypeScript + better-sqlite3
- 认证：JWT + bcryptjs

## 运行方式

```bash
npm install
npm run dev
```

前端启动在 http://localhost:5173，后端启动在 http://localhost:3001。

首次启动会自动创建 SQLite 数据库、建表并插入 3 条示例赛事数据。
